# Bank_Management_System

A console-based Bank Management System developed in Python using Object-Oriented Programming (OOP). This project allows users to create and manage bank accounts through a simple command-line interface while demonstrating core Python programming concepts.

Features

- Create a new bank account
- Secure login using account number and PIN
- Deposit money
- Withdraw money
- Transfer money between accounts
- Check account balance
- View transaction history
- Delete account
- Menu-driven interface

Technologies Used

- Python 
- Object-Oriented Programming (Classes & Objects)
- Dictionaries
- Lists
- Functions
- Loops and Conditional Statements
- File Handling (JSON support will be added in future updates)

Learning Outcomes

This project helped me practice:

- Python fundamentals
- Object-Oriented Programming
- Data structures (lists and dictionaries)
- Designing a medium-sized Python application
- Writing reusable and modular code
- Problem-solving and debugging

Future Improvements

- Store account data using JSON
- Split the project into multiple Python files
- Add exception handling for invalid input

How to Run

1. Clone the repository.
2. Open the project folder.
3. Run:

python main.py
